# KeyFlow

iPhone を MacBook の入力補助デバイスとして使う macOS アプリです。  
**テンキー**と**トラックパッド**の2モードを搭載しています。

---

## ダウンロード

[Releases](https://github.com/daisukenagashima-boop/keyflow/releases) から最新の `KeyFlow-x.x.x-universal.dmg` をダウンロードしてください。

---

## インストール

1. DMG を開いて `KeyFlow.app` を `/Applications` にドラッグ
2. 初回起動時に **「壊れている」エラーが出た場合**:
   ```bash
   xattr -cr /Applications/KeyFlow.app
   ```
3. KeyFlow を起動 → メニューバーにアイコンが現れる

---

## 初回セットアップ

### 1. アクセシビリティ権限を許可する

キーストロークとマウス操作を送るために macOS の権限が必要です。  
起動時に自動でガイド画面が表示されます。

1. 「設定を開く →」をクリック
2. 一覧に KeyFlow がある場合は **「−」で削除**
3. 「**＋**」で `KeyFlow.app` を追加してスイッチをオン
4. アプリに戻り「許可した ✓」をクリック（自動で確認されます）

> **更新後に権限が効かなくなった場合も同じ手順**で解決します（「設定を開く →」が古い登録を自動でリセットします）。

### 2. iPhone を接続する

1. Mac と iPhone を **同じ Wi-Fi** に接続
2. 画面に表示された QR コードを iPhone のカメラで読み込む
3. Safari が開き「● connected」と表示されれば完了

> iPhone のホーム画面に追加しておくと、次回からアイコンをタップするだけで起動できます（QR 読み込みは初回のみ）。

## 使い方

### テンキーモード

スプレッドシート（Excel / Google スプレッドシート / Numbers）の数値入力に最適です。

| 操作 | 動作 |
|------|------|
| ボタンをタップ | 対応するキーを Mac に送信 |
| ⌫ を長押し（0.6秒） | 入力フィールドを全削除（Cmd+A → Delete） |
| その他のキーを長押し | カーソル移動モード（指をスライドで上下左右に移動） |

```
%    /    +    -
7    8    9   Tab
4    5    6    ⌫
1    2    3
0   00    .   Enter
```

### トラックパッドモード

画面上部の「**トラックパッド**」タブをタップして切り替えます。

| 操作 | 動作 |
|------|------|
| 1本指でなぞる | マウスポインター移動 |
| 1本指タップ | 左クリック |
| 1本指長押し → そのまま動かす | ドラッグ |
| 2本指タップ | 右クリック |
| 2本指スワイプ | スクロール |

> 外部ディスプレイにも対応しています。

## トレイメニュー

メニューバーのアイコンを右クリックで以下の操作ができます:

- **QR コードを表示** — 接続用 QR を再表示
- **ログイン時に自動起動** — Mac 起動時に自動で常駐（デフォルトでオン）
- **アップデートを確認…** — 最新バージョンを手動チェック
- **終了** — アプリを終了

---

## アップデート

起動時に自動で新バージョンを確認します。新しいバージョンがあると通知が表示され、**「自動でインストール」** をクリックするだけで更新完了（再起動あり）。  
ダウンロード中は Dock アイコンにプログレスバーが表示されます。

---

## Android でも使える

QR コード画面には iPhone 用（`.local` URL）と Android 用（IP URL）の 2 種類が表示されます。Android の場合は Chrome で開いてください。

---

## セキュリティ

- **ローカル LAN 専用**。インターネットには公開しないでください
- 接続にはランダム生成されたトークンが必要です（URL に埋め込み済み）
- 受け付けるキー入力・マウス操作は限定されています（任意コマンド実行などは不可）

---

## トラブルシューティング

**QR コードが表示されない**
- トレイアイコンを右クリック → 「QR コードを表示」
- それでも出ない場合は「終了」してから再起動

**入力・操作が効かない**
- アクセシビリティ権限が切れている可能性があります
- 「設定を開く →」をクリックして KeyFlow を削除 → 再追加してください

**「壊れている」エラーが出る（Gatekeeper）**
```bash
xattr -cr /Applications/KeyFlow.app
```

**iPhone から繋がらない**
- Mac と iPhone が同じ Wi-Fi にいるか確認
- ルーターの AP Isolation（クライアント間通信遮断）をオフにする

**トラックパッドでポインターが動かない**
- アクセシビリティ権限が必要です（テンキーと同じ設定）

## 仕組み

```
[iPhone Safari]
     │ WebSocket (Socket.IO)
     ▼
[KeyFlow (Electron + Express) on Mac]
     ├─ キー入力 → execFile("osascript") → System Events → keystroke
     └─ マウス操作 → bin/mousecontrol (Swift) → CoreGraphics API
```

---

## ファイル構成

```
keyflow/
├── electron-main.js       # Electron メインプロセス・自動アップデート
├── server.js              # Express + Socket.IO + キー/マウス処理
├── latest.json            # アップデート確認用バージョン情報
├── bin/
│   └── mousecontrol       # マウス制御 Swift バイナリ（ユニバーサル）
├── src/
│   ├── index.html         # Mac 側ウィンドウ（QR表示・設定ガイド）
│   ├── style.css
│   └── preload.js
└── public/
    ├── index.html         # iPhone 側 UI（テンキー・トラックパッド）
    ├── style.css
    ├── app.js             # キー送信・トラックパッド・カーソルモード
    ├── sw.js              # Service Worker
    └── manifest.webmanifest
```
